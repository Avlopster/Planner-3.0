## Критерии приёмки шага

- По переменным окружения приложение переключается на PostgreSQL без изменения сценариев использования.
- Локальная разработка и текущий установщик по умолчанию остаются на SQLite.
- Репозиторий и миграции работают с обеими СУБД.
