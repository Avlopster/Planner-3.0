# Planner 3.0

Ветка разработки многопользовательского режима Planner. Основа — копия приложения Planner 2.0; вся новая функциональность (авторизация, аудит пользователей, поддержка PostgreSQL, серверное развёртывание) реализуется здесь, без изменений в основном проекте Planner 2.0.

## Назначение

- Серверная установка (Windows / Linux)
- Система авторизации пользователей
- Производительная СУБД (SQLite по умолчанию, опционально PostgreSQL)
- Логирование действий пользователей (аудит)

## Запуск

```bash
cd "Planner 3.0"
pip install -r requirements.txt
streamlit run Planner.py
```

Приложение будет доступно по адресу http://localhost:8501.

## Установка на Windows

В каталоге `installer/` подготовлен установщик Inno Setup с поддержкой **разностного обновления**: при повторной установке сохраняются конфигурация, БД и настройки; обновляются только файлы приложения и зависимости. См. [installer/README.md](installer/README.md).

## Изоляция от Planner 2.0

Данный проект имеет отдельный Git-репозиторий и не связан с репозиторием родительского проекта Planner. Изменения в «Planner 3.0» не затрагивают код и данные Planner 2.0.
