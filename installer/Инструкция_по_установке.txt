================================================================================
  PLANNER — ПОШАГОВАЯ ИНСТРУКЦИЯ ПО УСТАНОВКЕ
================================================================================

Требования: Windows 10/11 (64-bit). Устанавливать Python отдельно не нужно —
он входит в состав установщика.


ШАГ 1. Запуск установщика
-------------------------
• Найдите файл PlannerSetup.exe (или переданный вам установочный пакет).
• Запустите его двойным щелчком.
• При запросе контроля учётных записей (UAC) нажмите «Да» или «Продолжить».


ШАГ 2. Выбор папки установки
-----------------------------
• В окне мастера установки указана папка по умолчанию (например,
  C:\Program Files\Planner). При необходимости измените её.
• Нажмите «Далее» и примите условия (если отображаются).


ШАГ 3. Ожидание завершения установки
------------------------------------
• Установщик скопирует файлы, создаст виртуальное окружение и установит
  зависимости. Это может занять несколько минут.
• Обязательно включите пункт «Установить как службу Windows» — без него
  Planner не будет запускаться автоматически.
• На рабочем столе будет создан ярлык «Planner.url» для открытия приложения
  в браузере.
• В конце установки может быть предложено запустить службу Planner —
  при желании отметьте пункт и нажмите «Готово».


ШАГ 4. Первый запуск приложения
-------------------------------
• Дважды щёлкните по ярлыку «Planner» на рабочем столе.
• Откроется браузер по умолчанию со страницей приложения
  (http://localhost:8501).
• Если страница не открылась: убедитесь, что служба Planner запущена
  (службы Windows → Planner → Запустить).


ШАГ 5. Автозапуск после перезагрузки
------------------------------------
• После перезагрузки компьютера служба Planner запускается автоматически
  при загрузке Windows.
• Откройте приложение ярлыком на рабочем столе, как обычно.


УДАЛЕНИЕ ПРОГРАММЫ
------------------
• Откройте «Параметры» → «Приложения» → «Приложения и возможности»
  (или «Панель управления» → «Программы и компоненты»).
• Найдите «Planner» в списке и нажмите «Удалить».
• Подтвердите удаление. Служба Planner и ярлык с рабочего стола будут
  удалены автоматически.


При возникновении проблем проверьте, что антивирус или брандмауэр не блокируют
установщик и приложение, и что порт 8501 не занят другим ПО.

При обращении в поддержку можно приложить последний файл лога установки или
удаления из папки: %APPDATA%\Planner\Logs (файлы install_*.log, uninstall_*.log).

Логи работы приложения (planner_db.log, planner_actions.log, planner_errors.log) находятся в папке:
%LOCALAPPDATA%\Planner\Logs
